# Contributing to Optimizing-Phone-Android-Only

Thanks for your interest in this project! I welcome contributions to help make Android devices faster, lighter, and more efficient.

## How to Contribute

1. **Bug Reports:** If you encounter any issues, please open an issue in the "Issues" tab. Be sure to include your device model and Android version.
2. **Feature Requests & Improvements:** If you know of better optimization methods or lightweight apps, feel free to submit a "Pull Request" (PR).
3. **Documentation:** If you find any inaccuracies or want to improve the guide, suggestions and corrections are always welcome.

## Guidelines

* **Safety First:** Please ensure that any methods suggested are safe, ideally using "non-root" or "Shizuku" methods.
* **Minimalism:** Any recommended apps must be lightweight, privacy-focused, and free of bloatware.
* **Licensing:** By contributing to this project, you agree that your changes will be licensed under the **MIT License**.

## How to Submit a Pull Request

1. **Fork** this repository.
2. Create your feature branch (`git checkout -b feature/AmazingFeature`).
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`).
4. Push to the branch (`git push origin feature/AmazingFeature`).
5. Open a **Pull Request**.

Thanks for helping build a faster mobile experience! 🚀
